# 💻 온라인 서점 웹 서비스

---

### 🔍 Introduce

3학년 2학기 데이터베이스 과목 텀프로젝트

순천향대학교 온라인 도서관 시스템

### 🔨 Development Environment

- Front-End : React JS
- Back-End : Node JS
- Database : MongoDB

### ⚡️ Implementation Function

**Admin**

1. Product Upload
2. Customer Management
3. Product Management
4. Payment Management

**Client**

1. Paypal : Virtual Payment System
2. Buy Product
3. Shopping Basket
4. History
5. Profile
